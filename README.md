# Bulut Quinlan

Personal portfolio and resume site for a Principal Software Development Engineer in Test focused on release safety, production-risk intelligence, and high-confidence go/no-go decisions.

Live site: `https://bulut-sdet.vercel.app/`

## Professional Focus

I work at the point where software quality becomes business risk.

My background spans fintech and SaaS platforms where release confidence depends on more than passing tests. This site highlights my work in end-to-end automation, contract and event validation, observability-driven investigation, and release decision support across complex distributed systems.

## What This Site Covers

- principal-level SDET and QA leadership
- Playwright, Postman, Selenium, and API automation
- AWS event-driven systems validation
- Datadog and Splunk-based production investigation
- release readiness for high-risk workflows and integrations

## Built With

This is a static one-page site built with:

- HTML
- CSS
- vanilla JavaScript

## Repository Notes

- `index.html` contains the page content, styling, metadata, and interactive behavior
- `assets/resume.pdf` contains the downloadable resume
- `assets/brand-art.svg` is used for the favicon and social preview image

## Local Preview
